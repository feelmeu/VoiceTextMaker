# 0006 — GUI 界面开发

## 目标

开发完整的图形界面，支持文件选择、参数调节、进度显示、视频预览、历史记录。

## In Scope

- [ ] GUI 框架选型与搭建（Gradio / PyQt，ADR 决策）
- [ ] 文件选择面板（人物视频、参考音频、文本输入）
- [ ] 参数调节面板（语速、temperature、表情强度等）
- [ ] 生成进度显示
- [ ] 视频预览播放器
- [ ] 历史记录列表（可回看、重用）
